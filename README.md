# Music Platform Analytics

This project uses Matplotlib to visualize monthly platform visitor statistics and streaming trends for different music genres.

## Features

* Monthly visitor trend visualization
* Multiple line charts on a single subplot
* Custom colors, markers, and line styles
* Legends and axis labels
* Side-by-side subplot layout

## Technologies

* Python
* Matplotlib

## Visualization

The dashboard contains:

1. Monthly platform visitor counts
2. Monthly streams for EDM, Rock, and Hip-Hop genres

This project was created as part of learning data visualization with Python and Matplotlib.
